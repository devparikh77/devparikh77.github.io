# Data Science Portfolio

This is a static GitHub Pages portfolio site for showcasing data science projects.

## Publish on GitHub Pages

1. Create a GitHub repository named exactly:

   ```text
   your-github-username.github.io
   ```

2. Add these files to that repository.

3. Commit and push to the `main` branch.

4. In GitHub, open the repository and go to **Settings > Pages**.

5. Under **Build and deployment**, select:

   ```text
   Deploy from a branch
   ```

6. Choose:

   ```text
   main / root
   ```

With your GitHub username `devparikh77`, your site will be available at:

```text
https://devparikh77.github.io/
```

If your GitHub username is different, use `https://your-github-username.github.io/`.

## Where to edit

- `index.html`: Update your name, bio, resume link, project descriptions, and social links.
- `assets/css/styles.css`: Update colors, layout, and visual styling.
- `assets/img/portfolio-hero.svg`: Replace with a headshot or custom visual when ready.
